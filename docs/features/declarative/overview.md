# Declarative Models Overview

Define data models once with attributes; Lithair generates the rest.

## What you get

- CRUD endpoints with validation
- Persistence (events, snapshots, audit trail)
- Security hooks (permissions integration)
- Documentation and schema metadata

## Principles

- Single source of truth (struct + attributes)
- Explicit over implicit; readable attributes
- Forward-compatible schema evolution
